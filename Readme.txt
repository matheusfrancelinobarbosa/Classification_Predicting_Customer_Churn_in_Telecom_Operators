ML Project:
Predicting Customer Churn in Telecom Operators.

Target variable: 0 for 'no' and 1 for 'yes

In this project the Python programming language and its libraries were used.
Pandas and Numpy for data manipulation.
Matplotlib and seaborn for graphical visualization.
Sklearn for the ML models (LogisticRegression, RandomForest, KNN, Decision Tree and SVM).
